filter

结算每个元素得到布尔值，并返回布尔值为true的元素

例如：dataStream.filter { _ != 0}

keyBy

逻辑地将一个流分区，每个分区包含具有相同key的元素，在内部以hash的形式实现

注意：以下内容无法作为key

  • POJO类，但是没有重写hashCode()函数并且依赖Object的hashCode()实现
  • 任意形式的数组类型

Reduce

合并当前的元素和上次规约的结果，产生一个新的值

Fold

一个有初始值的分组数据流的滚动折叠操作，合并当前元素和前一次折叠操作的结果，并产生一个新的值。

例如：

val result: DataStream[String] = keyedStream.fold("fold")((str, i) => { str + "-" + i})

输入(1, 2, 3, 4, 5)，输出”fold-1”, “fold-1-2”, “fold-1-2-3”, …

NP难是什么意思？

  • 没有找到有效的解决方法，没能找到多项式时间复杂度解决。

货郎问题

NP难的问题，到现在都没有有效的解决方案

0-1背包问题

对于每个问题只有取或者不取，所以问题的解可以是一个0-1向量。

动态规划

多阶段决策过程，每步求解的问题是后面阶段求解问题的子问题，每步决策将依赖于以前的决策结果。

比如求解最短路径，不如就倒着来想这个问题。

生活中从苏州到重庆的最短路径，不如先去想武汉到重庆的最短路径，最后一步步倒推。

这种问题需要满足一个优化原则：最优子结构。

一个问题能否用动态规划，就是需要看，这个问题的最优解，里面的子问题是否都是对于子问题来说也是最优解的，简而言之，就是该问题是否满足具有最优子结构。
